# Study Workspace

This directory is reserved for the empirical study described in the
manuscript.

- `materials/` holds task instructions, condition prompts, assistant behavior
  specifications, and participant-facing procedures.
- `measures/` holds scale items, behavioral operationalizations, questionnaires,
  and data dictionaries.
- `analysis/` holds preregistration notes, analysis plans, and analysis code.
- `data/` is prepared for local raw and processed data; data contents are
  excluded from Git by default.

Avoid committing identifiable participant data or credentials.
